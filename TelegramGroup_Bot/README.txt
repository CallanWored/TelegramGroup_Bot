Бот удаляет из чата телеграм сообщения со ссылками. И пишет предупреждение

Инструкция
1. Создать бота в телеграм
2. Настроить файл TelegramGroup_Bot.ini
	2.1 Записать токен бота в Token_bot
	2.2 Записать Username из телеграм в User_exception, для которых бот будет делать исключения
3. Запустить TelegramGroup_Bot.exe
4. Добавить бота в группу
5. Сделать бота администратором группы
